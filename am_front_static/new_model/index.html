Un développement "Full Stack"

Les divers langages de programmation que maitrise un développeur sont primordiaux pour la richesse de ses créations. C'est pourquoi, pour créer votre site web, j'utilise une palette diverse d'outils performants me permettant de maitriser au mieux chaque étape du développement.





MDBootstrap

Le meilleur et le plus utilisé de tout les frameworks pour construire des sites web responsives. Il me permet de construire la partie front-end du site.

JavaScript

Langage de programmation de scripts indispensable pour créer du dynamisme dans une page web.

Django

Framework Python open source "pour les perfectionnistes avec des deadlines". Il constitue toute la partie backend de l'infrastructure du site.

MySQL

Système de gestion de base de données relationnelles. Il est le meilleur compromis dans son domaine, c'est avec ce systeme que je vais créer votre base de données si besoin.
